# gbin
My personal addition to /bin/
____
It contains the following commands :
- cell : displays a visualisation of a simple cellular automata. For more information, use the -h option.
- ezgh : a script that facilitates basic github operations. For more information, use the -h option.
- gbfi : a simple brainfuck interpreter. For more information, use the -h option.
- gobc : a base converter with a GUI, written in OCaml with the bogue graphic library.
- kaput : a short command to shut down your computer instantly (literally just "shutdown -h now")
- obc : a short command that compiles an ocaml program file (passed as an argument without the ".ml" extension), while specifying the use of the GUI library "bogue". I used it to make gobc.
- gup : a script that facilitates updating packages. It currently only features apt and opam, because I only use apt and opam.
- spaceBuster : a script that displays all the files that have spaces in their names in your current directory and its subdirectories (etc.). Useful when you want to "clean" your filesystem from all the mistakes you made on a GUI file manager...
- catclip : a shorthand for an xclip option, allowing you to pipe outputs into your clipboard : `foo | catclip`
